👋🏻 Hi, this is a Movie site, like Netflix

**_😵‍💫 With amazing features_**

1 - Multi language (English & Persian) 🌱<br />
2 - Full Responsive 💥<br />
3 - Live movie data powered by TMDB API 🎬<br />
4 - Hero slider with autoplay, fade effect & skeleton loading 💎<br />
5 - Search movies in real-time 🔍<br />
6 - Watch trailers (YouTube embed) 🎥<br />
7 - Download links per movie 👌🏻<br />
8 - User comments via Firebase Firestore 💬<br />
9 - Skeleton loading & lazy images across all pages ⚡<br />
10 - Modern dark color theme (Navy + Violet) 🎨<br />
11 - PWA support 📱<br />

---

### Tech Stack

- **Next.js** (Pages Router) with ISR & SSR
- **TMDB API** for live movie data
- **Firebase Firestore** for user comments
- **Tailwind CSS** with CSS variables for theming
- **Swiper.js** for sliders
- **react-i18next** for internationalization

---

### Environment Variables

Create a `.env.local` file with:

```env
TMDB_READ_ACCESS_TOKEN=your_token_here
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
```
